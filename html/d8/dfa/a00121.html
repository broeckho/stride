Detailed Description

template<typename E>
struct stride::util::Rn< E >::Info

POD representation of the RNManager's state.

If no state is available, i.e. state is an empty string, the initial state corresponding to the seed sequence is implied. If a state is available, the seed sequence is disregarded.
