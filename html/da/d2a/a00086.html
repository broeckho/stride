Detailed Description

Controls a geo-population generation run initiated with the command line interface (cli).

GeoPopController setup functions include (

  • checks the file system environment
  • interprets and executes the ouput prefix
  • intalls a stride logger
The GeoPopController execution:
  • creates a population (
Population)
  • creates a simulation runner (
SimRunner)
  • registers the appropriate viewers
  • runs the simulation
