Documentation

The documentation for the current version of the Stride project consists of:

  • Reference manual generated from the source using the doxygen tool.
  • User manual generated from latex sources and available in pdf format.
  • The installation information is provided in the INSTALL.txt file.
  • The file PLATFORMS.txt lists platforms on which the code was built and tested.
  • The file DEPENDENCIES.txt lists the softwares that the project depends on.

Following the installation of the project software with STRIDE_INCLUDE_DOC=TRUE, the Reference manual and User manual are available in the doc subdirectory of the installation directory. The other files are available in the root directory of the project software.
